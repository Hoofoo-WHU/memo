# Toast组件

## 引入
```js
import toast from '@/components/toast'
```
## 组件调用
### info
```ts
toast(msg: string[, time:number = 1000])
```

### success
```ts
toast.success(msg: string[, time:number = 1000])
```

### warn
```ts
toast.warn(msg: string[, time:number = 1000])
```

### error
```ts
toast.error(msg: string[, time:number = 1000])
```
